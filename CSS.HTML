BASICS OFCSS

CSS stands for CASCADING STYLE SHEETS.
CSS stands for Cascading Style Sheets
CSS describes how HTML elements are to be displayed on screen, paper, or in other media
CSS saves a lot of work. It can control the layout of multiple web pages all at once
External stylesheets are stored in CSS files

THREE WAYS TO ADDCSS

INLINE: INLINE CSS IS ADDED TO THE ELEMENTS DIRECTLY USING STYLE ATTRIBUTE
INTERNAL: INTERNAL CSS IS KEPT INSIDE THE HEAD TAG IN STYLE TAG
EXTERNAL TAG: IN OUR EVERYDAY LIFE EXTERNAL CSS IS GENERALLY USED.IT IS KEPT SEPERATELY INSIDE A .CSS STYLE SHEET

QUICK POINTS

BETWEEN INLINE AND EXTERNAL CSS MORE SIGNIFICANCE IS GIVEN TO THE ONE WRITTEN AT LAST
IF YOU WANT TO GIVE MORE IMPORTANCE TO BETWEEN THE TWO YOU CAN USE !important
WE CAN USE INSPECT ELEMENT/VIEW PAGE SOURCE TO EDIT HTML
